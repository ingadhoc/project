=========================
Project My Tasks By Stage
=========================

This module is very useful if all the projects share the same tasks stages and changes the way tasks are displayed in the sub-menu "My tasks".
Instead of showing the user his tasks grouped by personal stages, now it will display them grouped by the project's task stage.
